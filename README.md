# 포토카드 거래 서비스 프로젝트

## Period
  
  > 2023.10

## Explanation

- “덕질” 문화인 포토카드를 판매하고 구입할 때, 편리하게 포토카드 거래를 할 수 있는 프로젝트.
- Next.js, typescript, react 기반으로 프로젝트 생성
- tailwind css 사용으로 편리한 반응형 css 웹 뷰 개발
- 사진 편집 라이브러리인 fabric.js로 사진 편집 기능 개발
